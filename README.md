# 2D Fortnite Clone

A simplified 2D browser-based version of Fortnite built with HTML5 Canvas and JavaScript.

## Features

- Player movement with WASD keys
- Building system (press Q to toggle build mode)
- Shooting mechanics
- Health system
- Resource management (wood, brick, metal)
- Crosshair for aiming

## Controls

- WASD - Move player
- 1-5 - Switch weapons
- Q - Toggle build mode
- Left Click - Shoot/Build
- Right Click - Aim

## How to Play

1. Open `index.html` in your web browser
2. Use WASD to move your character
3. Press Q to enter build mode
4. Left click to shoot or build
5. Collect resources and build structures to protect yourself

## Development

To modify or enhance the game:

1. Edit `game.js` to modify game logic
2. Edit `style.css` to change the appearance
3. Edit `index.html` to modify the game structure
